# AI_Voice_Assistant_J.A.R.V.I.S

## Overview

Jarvis AI Voice Assistant is a Python-based desktop assistant that can perform various tasks through voice commands. The assistant listens to user instructions, processes them, and executes actions such as opening applications, searching the web, telling the current time, and providing information.

## Features

* Voice command recognition
* Text-to-speech responses
* Open websites and applications
* Web search functionality
* Current time and date announcements
* Play music or media files
* Basic conversational interactions
* Extensible command structure for adding new features

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* PyAudio
* Webbrowser
* Datetime
* OS Module

## Installation

1. Clone the repository:

```bash
git clone https://github.com/shreyashnikam25/jarvis-ai-assistant.git
```

2. Navigate to the project folder:

```bash
cd jarvis-ai-assistant
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python main.py
```
